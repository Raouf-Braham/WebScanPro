## About the Project

<!-- Mission -->
### Mission
To simplify and automate web security testing, making it accessible to everyone. WebScanPro empowers users to identify and mitigate security risks through intuitive dashboards, modular tool integration, and reliable reporting ‚Äî without requiring deep expertise in cybersecurity.

<!-- Tutorial -->
### Tutorial

<p>New to WebScanPro? Here's how to get started: First, sign up / login as a client or admin. Second, launch a Scan via the dashboard using tools like Nmap, Subfinder, or ZAP.
  Each tool's simulation is built to give users real-time feedback and a hands-on experience.
</p>
